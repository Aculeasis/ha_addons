# Aculeasis Home Assistant Add-ons

Custom add-ons for [Home Assistant](https://www.home-assistant.io/). Proxy tools, VPN clients, and monitoring utilities.

## Add-ons

| Add-on | Description |
|--------|-------------|
| **[gost-ss](gost-ss/)** | Shadowsocks & GOST proxy runner. Run multiple ss-local, ss-server, or GOST instances. |
| **[privoxy](privoxy/)** | Non-caching web proxy with ad blocking and privacy filtering. |
| **[adguardvpn](adguardvpn/)** | AdGuard VPN as a SOCKS5 proxy. Route traffic through AdGuard VPN servers. |
| **[xray](xray/)** | Xray-core proxy (VLESS, VMess, Trojan, Shadowsocks). |
| **[proxy-monitor](proxy-monitor/)** | Real-time SOCKS5 proxy health monitoring dashboard with TCP/UDP checks. |
| **[remote-smart](remote-smart/)** | Storage device S.M.A.R.T. monitoring via MQTT for Home Assistant. |

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**.
2. Click the menu (⋮) → **Repositories** → **Add repository**.
3. Paste: `https://github.com/Aculeasis/ha_addons`
4. Click the new repository, then install the add-ons you need.
