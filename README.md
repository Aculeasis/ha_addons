# Home Assistant Add-ons Repository

This is a custom Home Assistant add-on repository.

## Add-ons

* **[gost-ss](gost-ss/)**: GOST / Shadowsocks client and proxy add-on.
* **[privoxy](privoxy/)**: Privoxy non-caching web proxy add-on.
* **[xray](xray/)**: Xray-core proxy add-on.

## Installation

1. Go to **Settings** > **Add-ons** > **Add-on Store** in your Home Assistant.
2. Click the menu (three dots) in the top right corner and select **Repositories**.
3. Add the URL of this repository.
4. Refresh and install the desired add-ons.
