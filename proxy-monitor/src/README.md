# Proxy Monitor

Real-time SOCKS5 proxy monitoring dashboard.

## Features
- TCP/UDP checks + external IP detection
- Live WebSocket dashboard with glassmorphism UI
- Detailed historical charts (Chart.js)
- SQLite storage with auto-retention
- Web-based configuration

## Development
```bash
pip install -r requirements.txt
python server.py
```

See [DOCS.md](../DOCS.md) for configuration details.
